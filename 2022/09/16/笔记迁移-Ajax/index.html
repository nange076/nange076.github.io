Ajax可以通过不刷新页面实现与后端的数据交互，能够很好地提高用户体验

Ajax的基本使用步骤

  1. 创建XMLHttpRequest对象
  2. 初始化，设置请求类型和请求的URL地址 、设置请求头、设置响应类型
  3. 发送 ，如果是POST请求可以发送数据
  4. 事件绑定
    1. 判断readyStatus：5个状态；0：默认值 ，1：初始化，2：发送，3：接收到部分响应数据，4：接收到全部响应数据
    2. 判断status（响应字符串）；200(ok) 4xx 5xx ….
    3. 获取响应内容（response)

获取JSON格式的response

有两种获取方式

自动转换

需要设置响应格式，使用xhr.responseType="json";设置

手动转换

使用JSON.parse()将接收的数据转换为JSON（注意：即使后端发送的数据就是JSON也需要转换）

后台使用JSON.stringify()发送JSON数据

使用nodemon简化后端修改的启动

修改后端代码后需要频繁重启服务，使用nodemon来自动进行重启

安装： npm install -g nodemon

启动服务端： nodemon server.js

ps：安装后启动服务如果显示系统上禁止运行脚本，管理员权限打开powershell 输入set-ExecutionPolicy RemoteSigned -输入y，解决问题
